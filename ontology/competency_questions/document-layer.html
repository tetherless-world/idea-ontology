Document Structure & Metadata Competency Questions

Competency questions for querying document structure, discourse elements, authorship, publication metadata, and citations in scholarly works.

CQ-DOC-01: What paragraphs does a given scholarly work contain?

Intent

Retrieve all structural paragraphs that are part of a scholarly document.

Natural Language Question

What paragraphs does this scholarly work contain?

SPARQL Query


PREFIX doco: 
PREFIX idea: 
PREFIX po:   

SELECT ?sec ?para
WHERE {
  idea:work-semsur-2018 po:contains ?sec .
  ?sec po:contains ?para .
}

CQ-DOC-02: What discourse elements appear in this document?

Intent

Identify the discourse roles instantiated within a scholarly document.

Natural Language Question

What discourse elements (Introduction, Methods, Results, etc.) appear in this document?

SPARQL Query


PREFIX rdf:  
PREFIX rdfs: 
PREFIX po:   
PREFIX deo:  
PREFIX idea: 

SELECT ?sec ?secType
WHERE {
  idea:work-semsur-2018 po:contains ?sec .
  ?sec rdf:type deo:DiscourseElement ;
       rdf:type ?secType .
  ?secType rdfs:subClassOf* deo:DiscourseElement .
}

Expected Result


Evaluation of SemSur ontology | deo:Evaluation

SPARQL Query


PREFIX rdf:   
PREFIX rdfs:  
PREFIX fabio:

SELECT ?paperType
